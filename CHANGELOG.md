# Changelog

## [0.2] - 2025-03-30

### Новые инструменты
- **vision** — анализ изображений мультимодальной моделью Ollama
  - Автоматическая валидация и конвертация изображений в JPEG
  - Preflight-проверка перед отправкой в Ollama
  - Проверка Content-Type для веб-изображений
- **curl** — загрузка файлов и работа с HTTP-запросами
- **web_extract** — извлечение ресурсов с веб-страниц
- **image_ascii** — ASCII-генератор изображений
- **--update** — инструмент обновления версии botinok

### Новые возможности
- Поддержка macOS
- Режим "только чат" для моделей без поддержки инструментов
- Пользовательские конфиги (в дополнение к проектным)
- Debug-режим для отладки
- Системный промт для управления поведением
- Промты вынесены в отдельную папку `prompts/`, клонируются внутрь сессии
- Опасный режим для file_system инструмента

### Улучшения
- Работа визарда при Ollama через nginx
- Оптимизация работы через SSL с Ollama
- Уменьшение мерцания в SSH-терминалах с низкой скоростью
- Компактный прогрессбар контекста
- При возврате в сессию сразу выводится последний ответ
- Улучшение автообновления с поддержкой Python-зависимостей

### Исправления
- Исправлена ошибка с переменной `ans` в file_system
- Исправлена обработка изображений с неизвестным форматом
- Исправлена валидация изображений: preflight-проверка
- Исправлена обработка ошибок сохранения конфигов

### Прочее
- Расширен скрипт установки для RHEL-систем
- Добавлен логотип в README
- Обновлена документация

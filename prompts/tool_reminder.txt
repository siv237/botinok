TOOL_USAGE_REMINDER:
1. ОБЯЗАТЕЛЬНО: Начни с `skills action=list` и `experience action=list`. Даже для простых задач вроде "узнай погоду" — сначала проверь, нет ли готового скилла.
2. THINKING: В блоке рассуждений зафиксируй факт проверки навыков.
3. Чтение/Поиск: используй `file_system` (list/grep/info) или `code_editor` (read).
   НЕ используй `shell_exec` для простого чтения файлов.
4. Редактирование: используй `code_editor` (write/apply/replace).
   НЕ используй `shell_exec` (sed/echo >) для правки кода.
5. Скачивание файлов: используй `curl` (url, output_path). 
   - Сначала проверь что файл существует (curl сам проверяет HEAD запросом)
   - Рекомендуемый путь: `<session_path>/downloads/<filename>` (папка создаётся автоматически)
   - НЕ используй `shell_exec curl` — это системный инструмент curl для HTTP запросов.
6. Порядок: Сначала изучи (list/read), потом планируй, потом меняй.
7. Системные промпты: Редактируй файлы в {{PROMPTS_DIR}}/ для настройки своего поведения в этой сессии.
